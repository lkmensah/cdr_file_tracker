# **App Name**: File Tracker

## Core Features:

- Correspondence Tracking: Logs new incoming correspondences (letters, documents, court processes) by capturing metadata (date, file number, suit number, subject, movement log, attorney, document number, remarks).
- File Matching: Automatically suggest existing files based on the captured metadata of the correspondence, facilitating quick assignment to the correct file, using fuzzy search as a tool.
- New File Creation: If no matching file is found, creates a new file entry in the database with the provided metadata.
- File Detail View: Displays all metadata associated with a selected file, providing a comprehensive view of its contents and history.
- Search and Filter: Enables users to search and filter files based on any metadata field (e.g., date, file number, attorney).

## Style Guidelines:

- Primary color: Light, desaturated blue (#A7D1AB), reflecting trustworthiness and organization.
- Background color: Very light gray (#F0F0F0), for a clean and professional feel.
- Accent color: Muted teal (#69A297) to highlight important actions and information.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Simple, professional icons to represent different file types and actions.
- Clean, structured layout with clear visual hierarchy to ensure ease of navigation and information retrieval.
- Subtle animations for transitions and feedback to improve user experience without being distracting.
